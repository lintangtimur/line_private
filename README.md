# LINE BOT AUTOREAD
This bot can Autoread message from personal chat



## Requirement
- Python version 3.6
- line-py
- var_dumper
## How to use
1. Copy content from .env.example to .env file
2. Fill that format, ADMIN_MID is your userid
3. run with `python s.py`
